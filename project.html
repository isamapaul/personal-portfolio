PAU Cafeteria Management System

Problem

Students often faced long queues and delays when ordering food on campus.

Solution

I built a digital cafeteria system that allows students to place orders online while staff efficiently manage and track orders.

Key Features

  • Online food ordering
  • Admin dashboard
  • Order tracking

Technologies Used

Java, MySQL, HTML, CSS

What I Learned

This project strengthened my understanding of system design, database integration, and building user-friendly interfaces.
